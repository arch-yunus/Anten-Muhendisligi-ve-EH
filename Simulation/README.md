# Elektromanyetik Simülasyon Çekirdeği (Simulation Core) 💻🛰️

Bu modül, teorik tasarımların sayısal olarak doğrulanması için gereken araçları içerir.

## 📂 İçerik
- **[Python NEC Wrapper](Python_NEC_Wrapper.py):** NEC2++ motoru için otonom giriş dosyası (.nec) üreticisi ve yürütücüsü.
- **[Workflow Guide](Workflow.md):** Standart EM tasarım döngüsü rehberi.

## 🚀 Kullanım
NEC sarmalayıcısını kullanarak bir dipol anten simülasyonu başlatmak için:
```bash
python Simulation/Python_NEC_Wrapper.py
```
